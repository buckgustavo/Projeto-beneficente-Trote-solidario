# 🤝 Trote Solidário — UniFACEF 2026

> **"Colecionando memórias de Solidariedade"**  
> Projeto web institucional do Trote Solidário do Centro Universitário Municipal de Franca (Uni-FACEF), edição especial de 75 anos.

***

## 📋 Sobre o Projeto

O **Trote Solidário Uni-FACEF** é uma iniciativa institucional presente na universidade há mais de 30 anos (desde 1993), que visa à inserção do discente no ambiente acadêmico, focando a promoção social e agregando em sua formação a importância de esforços que tornem suas decisões mais éticas e humanas.

Esta aplicação web foi desenvolvida para organizar, divulgar e gerenciar todas as informações da edição 2026, que celebra os **75 anos do Uni-FACEF** e está alinhada aos **Objetivos de Desenvolvimento Sustentável (ODS)** da ONU.

### ✨ Pilares do Evento

| Pilar | Descrição |
|---|---|
| 🎓 Formação Cidadã | Capacitar cidadãos para a vida profissional e comunitária pelo saber e pelo ser |
| 🤝 Solidariedade | Participar da busca de soluções para problemas da comunidade local e regional |
| ⚖️ Ética & Cidadania | Promover o bem comum e o progresso social através de ações éticas e humanas |

***

## 🌐 Páginas da Aplicação

O projeto é uma aplicação web estática multi-página composta pelos seguintes arquivos:

| Página | Arquivo | Descrição |
|---|---|---|
| 🏠 Início | `index.html` | Página principal com visão geral do evento, datas e coordenação |
| 📖 Introdução | `introducao.html` | Apresentação e contexto do Trote Solidário |
| 📐 Formato | `formato.html` | Regras e formato geral do evento |
| 👥 Equipes | `equipes.html` | Listagem e informações das 9 equipes participantes |
| 🏆 Provas & Metas | `provas.html` | Detalhamento das provas e metas de arrecadação |
| 🔗 Integração | `integracao.html` | Atividades e dinâmicas de integração entre calouros |
| 📅 Cronograma | `cronograma.html` | Linha do tempo e datas do evento |
| 🧮 Calculadora | `calculadora.html` | Calculadora de horas-atividade por doação |
| 🃏 Figurinhas | `figurinhas.html` | Álbum digital de figurinhas do evento |

## ⏱️ Horas-Atividade

Os alunos podem conquistar até **60 horas-atividade** por meio de suas contribuições:

- 📦 **30 horas** — Kit Alimentação (completo) + ½ Suplemento Alimentar
- 🩸 **20 horas** — Doação de Sangue (comprovante necessário)
- 📈 Dobrando a doação → carga horária dobrada (limite máximo: 60h)
- 📜 Certificados disponibilizados no Portal em até **10 dias úteis** após a entrega da lista

## 🚀 Como Executar

Por ser uma aplicação web estática, não requer instalação de dependências ou servidor backend.

### Opção 1 — Abrir diretamente no navegador

```bash
# Clone o repositório
git clone https://github.com/buckgustavo/Projeto-beneficente-Trote-solidario.git

# Acesse a pasta
cd Projeto-beneficente-Trote-solidario

# Abra o arquivo principal no navegador
open index.html          # macOS
xdg-open index.html      # Linux
start index.html         # Windows
```

### Opção 2 — Servidor local (recomendado)

```bash
# Com Python 3
python3 -m http.server 8000

# Com Node.js (npx)
npx serve .
```

Acesse: `http://localhost:8000`
